# To-Do-List

### Features:
- [x] Adicionar tarefa
- [x] Excluir tarefa
- [x] Marcar tarefa concluida ou desmarcar tarefa.

### Tecnologias:

- Reacjs
- JavaScript

```bash
git clone https://github.com/Feliper-Silva/tolist
```
- Yarn
```bash

```
- Npm
